# 立项档案（intake/）

新项目从受理到建壳的完整记录——一项目一份，记录受理要素、建壳回执、首推记录。

**命名规范**：`{项目id}.md`

**模板结构**：
- 立项要素（项目 id / 名称 / 定位 / 技术栈 / 远端仓库地址 / 分类 / 关联项目）
- 建壳回执（install-downstream.sh 产出时间、自检结果）
- 首推记录（git init + commit + push 时间、远端地址）
- PROJECTS/STATUS 登记行（照 coordination 真源引用）
